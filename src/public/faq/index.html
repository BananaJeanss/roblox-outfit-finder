Frequently Asked Questions

What does this site do?
Roblox Outfit Finder lets you search for any Roblox user's saved outfits by entering their username or user ID.
This only includes user-saved outfits. Default/catalog outfits are excluded from the results.

How do I search for a user?
Enter the user's Roblox username, or use id:USERID (e.g. 2837719) to search by user ID, then click "Find outfits".

Why do I see a "ratelimited" or error message?
The Roblox API has strict rate limits. If you see a ratelimit message, wait about 15-20 seconds and try again.

Why are some outfits missing images?
Sometimes Roblox is still generating the outfit thumbnail, or the image is blocked. Try again later if you see "Loading..." or "Blocked".

How long is data cached?
User and outfit data is cached for 5 minutes to speed up repeat lookups and reduce API calls.
